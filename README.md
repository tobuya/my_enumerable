## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need to have `Ruby` installed on your computer

### Setup

Clone this repository to your desired folder:

```sh
  cd my_folder
  git clone git@github.com:tobuya/my_enumerable.git
```

### Install [optional]

Install inters for this project with:

```sh
  cd my_enumerable
  bundle install
```

### Usage

To run the project, execute the following command:

```sh
  irb -r ./my_list.rb
```
